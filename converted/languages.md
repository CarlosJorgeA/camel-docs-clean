# Languages
:::

:::::::::: {#content}
::::: {#preamble}
:::: sectionbody
::: paragraph
To support flexible and powerful [Enterprise Integration
Patterns](components:eips:enterprise-integration-patterns.html), Camel
supports various Languages to create an [Expression](expression.html) or
[Predicate](predicate.html) within the [Routes](routes.html) and
[DSL](dsl.html)..
:::
::::
:::::

:::::: sect1
## Supported languages {#_supported_languages}

::::: sectionbody
::: paragraph
There are about 20 different
[Languages](components:languages:index.html) such as scripted
programming languages like Groovy, and template based languages like
Velocity and Freemarker, and XML/JSon languages, and many others.
:::

::: paragraph
Most of these languages are also supported used as [Annotation Based
Expression Language](parameter-binding-annotations.html) in Java beans.
